# SQL Practice – Data Handling Fundamentals

## Overview

This repository contains my SQL practice focused on building a strong foundation in data handling and querying.  

Understanding how to query and analyze data helps bridge the gap between marketing strategy and data-driven decision-making.  
These exercises focus on practical SQL skills such as retrieving, filtering, and sorting data—skills that support insights from customer behavior, sales trends, and campaign performance.

---

## Purpose

- Strengthen understanding of database fundamentals
- Practice writing clear and readable SQL queries
- Learn to retrieve, filter, sort, and limit data effectively
- Think about data in real-world scenarios 

---

## Topics Covered

- `SELECT` / `FROM`
- `WHERE`
- `ORDER BY` / `LIMIT`
- `IS NULL` / `IS NOT NULL`
- `AND` / `OR`
